# EstateOpus - Real Estate Landing Page

This is a Next.js application for a real estate website landing page, built with Firebase Studio.

## App Features

- **Property Listings**: Showcases real estate properties.
- **Interactive Map**: A section to explore property locations.
- **AI-Powered Recommendations**: An AI tool to help users find their dream home.
- **Contact Form**: For user inquiries.
- **Agent Profiles**: Displays real estate agents.
- **Testimonials**: Client testimonials section.

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

You can start by editing the page at `src/app/page.tsx`.
The placeholder data for properties, agents, and testimonials can be found in `src/app/data/`.
All placeholder images are defined in `src/lib/placeholder-images.json`.
